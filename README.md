**Prono Challenge** est l'application ultime pour vivre la Coupe du Monde 2026 entre amis. Transformez chaque match en compétition privée, prédisez les scores exacts, et dominez le classement de votre groupe.

👉 **[Accéder à la Web App & Téléchargements](https://fnnktkygl-code.github.io/mondial_2026/)**

---

## 🌟 Fonctionnalités Clés

### 🎯 Pronostics en Temps Réel
*   Prédisez le score exact de chaque match (104 matchs au total).
*   **Verrouillage automatique** : Les pronostics se ferment dès le coup d'envoi pour garantir l'équité.
*   Système de points dynamique (Score exact vs Bon vainqueur).

### 👥 Compétition Sociale
*   **Groupes Privés** : Créez votre ligue et invitez vos amis avec un code unique.
*   **Classement Global** : Comparez vos talents d'analyste avec le monde entier.
*   Profils personnalisés avec avatars et choix de l'équipe favorite.

### 📊 Expertise & Data
*   **Probabilités FIFA** : Consultez les chances de victoire basées sur l'index officiel.
*   **Cotes de Titre** : Suivez l'évolution des favoris pour le sacre final.
*   **Tableau (Bracket)** : Visualisez tout le parcours de la phase finale.

### 🚀 Gamification
*   **Joker Booster** : Doublez vos points sur un match stratégique (1 par tour).
*   **Badges & XP** : Montez en niveau et obtenez le rang de "Special One".
*   **Alertes Intelligentes** : Recevez une notification 1h avant les matchs pour ne jamais rater un prono.

## 📱 Multi-Plateforme

L'application est conçue pour être accessible partout :

1.  **Web App** : Jouez instantanément sur votre navigateur sans installation.
2.  **Android** : APK disponible avec système de mise à jour automatique (OTA) sans perte de données.
3.  **macOS** : Application desktop pour une expérience fluide sur ordinateur.

## 🛠 Architecture Technique

*   **Frontend** : [Flutter](https://flutter.dev) (Dart) pour une UI haute performance et cohérente.
*   **Backend** : [Firebase](https://firebase.google.com) (Firestore & Auth) pour la synchronisation temps réel.
*   **Distribution** : GitHub Pages (Landing Page) & GitHub Releases (Binaires).
*   **Design** : Typographie **Syne** & **Inter**, thématique "Dark Mode" premium.

## 📦 Installation & Développement

### Prérequis
*   Flutter SDK (dernière version stable)
*   Un projet Firebase configuré

### Lancement local
```bash
# 1. Récupérer les dépendances
flutter pub get

# 2. Lancer l'application
flutter run
```

### Build pour la production
```bash
# Web
flutter build web --base-href "/mondial_2026/"

# Android (APK)
flutter build apk --release

# macOS
flutter build macos --release
```
